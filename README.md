# TaskMaster

A simple and clean task management web app built with HTML and CSS, focused on structure, layout, and user-friendly design.

This project showcases my ability to build responsive interfaces and organize UI elements effectively without relying on frameworks.

## 🚀 Live Demo
https://taskmaster-indol-five.vercel.app

## 🧰 Tech Stack
HTML · CSS

## ✨ Features
- Clean and minimal task layout
- Structured UI for task organization
- Responsive design for different screen sizes
- Beginner-friendly and lightweight implementation

- `taskmaster.css` — Styling and layout
